## Project Overview

This project analyzes a dataset of 1,200 teenagers to understand the correlation between their daily social media habits and their mental well-being (stress, anxiety, depression). The goal of this project is to clean, query, and visualize the data to draw actionable insights regarding screen time.

## Objectives

  • Data Querying: Write SQL queries to extract behaviors and aggregate mental health scores.
  • Data Visualization: Build a visually clean and appropriate HTML/JS dashboard to present the findings.
  • Transformation: As a Data Analyst - translate raw metrics into understandable human behaviors.

## Key Data Insights

  • The Sleep Deficit: The average sleep time for teens is 6.4 hours. High pre-sleep screen time (2+ hours) correlates to significantly lower sleep duration.
  • Platform Averages: Instagram is the most prominent standalone platform, but 33% of users use both TikTok and Instagram heavily.
  • Stress Correlation: Average stress rests at 5.4/10. Heavy social media usage (7+ hours/day) correlates with higher anxiety metrics and lower academic performance.
